# 简化版显示设置功能说明

## 功能概述

已经按照您的要求简化了显示设置界面，现在只包含以下核心功能：

### 1. 窗口大小调整
- **预设尺寸**: 提供4个常用尺寸选项
  - 800 x 600 (小)
  - 1000 x 700 (中等) 
  - 1200 x 800 (大)
  - 1400 x 900 (超大)
- **自定义尺寸**: 可以精确设置像素值
- **立即生效**: 点击"应用"按钮立即调整窗口大小

### 2. 启用自适应
- **自适应缩放开关**: 根据屏幕分辨率和DPI自动调整
- **需要重启**: 更改此设置需要重启应用程序

### 3. 字体大小调整
- **字体级别**: -3到+3级调整
- **立即生效**: 在非自适应模式下立即应用
- **持久保存**: 设置会保存到文件中

## 使用方法

### 访问设置
通过主窗口菜单：**设置 > Display**

### 即时调整模式（推荐）
1. **关闭**"启用自适应缩放"
2. 选择合适的窗口大小或设置自定义尺寸
3. 调整字体大小级别
4. 点击"**应用**"按钮立即生效
5. 点击"**确定**"保存并关闭

### 自适应模式
1. **开启**"启用自适应缩放"
2. 点击"**确定**"
3. 选择"**是**"重启应用程序
4. 重启后窗口会根据屏幕自动调整

## 设置保存

- 所有设置自动保存到 `user_settings.json` 文件
- 程序重启后会恢复上次的设置
- 非自适应模式的更改立即生效
- 自适应模式的更改需要重启

## 主要改进

✅ **删除了复杂的配置选项** - 只保留实用的核心功能
✅ **立即应用功能** - 除自适应外，其他设置点击应用立即生效  
✅ **简化的界面** - 更直观的操作方式
✅ **设置持久化** - 自动保存用户偏好
✅ **智能提示** - 清楚告知哪些设置需要重启

